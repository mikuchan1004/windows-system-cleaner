불필요한 임시 파일들을 안전하게 정리해주는 프로그램입니다.

> [!CAUTION]
> ### 🛡️ 백신 오진 안내 (False Positive)
> 단일 파일(`.exe`) 패키징 구조와 시스템 임시 파일 정리를 위한 관리자 권한 요구로 인해, 일부 특정 백신(**Bkav Pro**)에서 오진으로 인한 경고(1건)가 발생할 수 있습니다.
> 
> **Microsoft Defender**를 포함한 국내외 주요 메이저 백신 검사는 모두 안전하게 통과(Clean) 완료되었습니다. 본 프로그램은 모든 소스 코드가 투명하게 공개된 오픈 소스이며, 어떠한 악성 코드도 포함하고 있지 않으니 안심하고 사용하셔도 됩니다.

> [!NOTE]
> **Microsoft Defender 검사 완료**
> 본 프로그램은 Windows Defender에서 안전함(Clean)을 확인받았습니다. 디지털 서명이 없는 개인 제작 도구 특성상 일부 외부 백신에서 오진이 발생할 수 있으나, 소스 코드가 공개되어 있으므로 불안하신 분은 직접 검토 후 빌드하여 사용해 주시기 바랍니다.

## 🛠 주요 기능
- **비주얼 스튜디오 청소**: 개발 중 발생하는 `bin`, `obj`, `.vs` 폴더 등 불필요한 빌드 아티팩트 제거
- **디스코드 캐시 정리**: 쌓여있는 Discord 이미지 및 동영상 캐시 파일을 삭제하여 용량 확보
- **시스템 최적화**: 윈도우 임시 폴더(Temp), 프리패치(Prefetch), 업데이트 캐시 등 시스템 쓰레기 파일 소탕
- **안전한 삭제**: 사용 중인 파일은 건너뛰고 삭제 가능한 파일만 안전하게 처리

## 🚀 사용 방법
1. [릴리즈](https://github.com/mikuchan1004/windows-system-cleaner/releases)에서 최신 버전의 실행 파일(`.exe`)을 다운로드합니다.
2. 프로그램을 실행하고 청소 버튼을 눌러 여유 공간을 확보합니다.
